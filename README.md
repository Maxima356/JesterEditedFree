# JesterEditedFree

![Lethal Company](https://img.shields.io/badge/Lethal%20Company-Mod-orange?style=for-the-badge)
![Version](https://img.shields.io/badge/version-1.2.5-blue?style=for-the-badge)
![Thunderstore](https://img.shields.io/badge/Thunderstore-Download-success?style=for-the-badge)

A custom sound pack for the Jester enemy in Lethal Company, replacing the default music box sounds with meme music.

## 🎵 Features

When the Jester begins to turn its crank, this mod plays:
- **Intro sound**: *Hava Nagila* (original)
- **Random cranking sounds**: One of two tracks is randomly selected when you launch the game
  - [Red Sun in the Sky](https://youtu.be/_TiSUBMCJ90)
  - [Ching Cheng Hanji](https://youtu.be/VN-DG5MgDy0)
  - [Arabian Psycho](https://youtu.be/A7Pv8Tg4C2Q)

> **Note:** The sound variant is now synchronized with the Map Seed. This means all players in the same lobby will hear the exact same sound. The audio files are loaded dynamically when the Jester spawns.

## 📦 Installation

### Prerequisites
- [BepInEx](https://thunderstore.io/c/lethal-company/p/BepInEx/BepInExPack/) installed
- [LCSoundTool](https://thunderstore.io/c/lethal-company/p/no00ob/LCSoundTool/) mod (required dependency)

### Steps
1. Download the latest version from [Thunderstore](https://thunderstore.io/c/lethal-company/p/Maxima356/JesterEditedFree/)
2. Extract the archive
3. Copy the contents to your BepInEx plugins folder:
   ```
   Lethal Company/BepInEx/plugins/JesterEditedFree/
   ```
4. Launch the game

## 🛠️ Technical Details

- **Audio format**: `.ogg`
- **File structure**:
  ```
  JesterEditedFree/
  ├── JesterFree.dll
  ├── intro.ogg        # Hava Nagila
  ├── solo_01.ogg      # Red Sun in the Sky
  └── solo_02.ogg      # Ching Cheng Hanji
  └── solo_03.ogg      # Arabian Psycho
  ```
